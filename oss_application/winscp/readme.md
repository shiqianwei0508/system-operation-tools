use winscp from command line
=============================

use winscp from CLI
-----------------------------

```
"C:\Program Files (x86)\WinSCP\WinSCP.com" /script=winscp.txt /xmllog="retrieve_logs.log"
```


use private key on winscp
--------------------------
1. Run WinSCP
2. Tool -> Run PuTTYgen
3. Load RSA-Key
4. Put "Save private key"
